# Gameplay
### BP
The main idle game. Buy generators that generate BP. Use the BP to buy more generators. Climb the BP ladder!
- Features a global leaderboard
- Live balance updates
- BP Generator shop
- Mining
### Items
Collectable items that help you get more BP
- Consumables
- Tools
- Cosmedics (WIP)
- Lootboxes
### Pickaxes (WIP)
Used for mining
